MetaQL: Meta Query Language


MetaQL provides a Domain Specific Language (DSL) for database queries, providing an abstraction across NoSQL, SQL, Graph, and Apache Spark. This allows the same query to be executed regardless of the backend implementation.

Vital AI provides an implementation of the MetaQL specification and connectors to a variety of databases including: MySQL and MySQL compatible, Amazon Redshift, DynamoDB, Allegrograph Sparql Triplestore, and Apache Spark-SQL (Dataframes).
